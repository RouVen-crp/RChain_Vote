# RChain Vote Project

在你希望的作为项目根目录的文件夹中使用如下命令
```
git clone https://github.com/RouVen-crp/RChain_Vote.git
```
然后再进行下列的Project Setup步骤

## Project setup

```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install

# pnpm
bun install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev

# bun
bun run dev
```

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build

# pnpm
pnpm build

# bun
bun run build
```

### Lints and fixes files

```
# yarn
yarn lint

# npm
npm run lint

# pnpm

# bun

```
